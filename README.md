# ระบบบัญชีรายรับรายจ่าย

สวัสดีครับขอแนะนำตัวอีกครับนะครับผม นายอธิชา จันทร์พฤกษ์(พีพี) ครับ โปรแกรมนี้เป็น "ระบบจัดการรายรับ-รายจ่าย" ที่ผมพยายามพัฒนาขึ้นมาตามโจทย์ที่ทางบริษัทส่งมาให้ครับ

## แนวคิดและจุดประสงค์
แนวคิดของผมก้คือว่า เป็นโปรแกรมที่ช่วยให้เราสามารถบันทึกรายการเงินเข้าและเงินออก เลือกหมวดหมู่ได้ และดูสรุปผลย้อนหลังได้ครับ โดยเน้นที่ความเรียบง่ายและการใช้งานที่ครอบคลุมความต้องการพื้นฐานทั้งหมด แถมยังสามารถสรุปผลออกมาเป็นรูปแบบไฟล์ csv, json และ excel ได้อีกด้วยครับ

## ฟีเจอร์ที่พัฒนาสำเร็จแล้ว
จนถึงตอนนี้ผมได้พัฒนาระบบหลักๆ โดยมีฟีเจอร์ดังนี้ครับ

### 1. ระบบสมาชิก (Authentication)
*   สมัครสมาชิกและเข้าสู่ระบบด้วย Username
*   มีการเก็บรหัสผ่านอย่างปลอดภัย (hash ด้วย bcrypt)
*   มี Middleware ตรวจสอบสิทธิ์การเข้าถึงข้อมูล เพื่อป้องกันไม่ให้คนอื่นเข้าถึงข้อมูลของเราได้ครับ

### 2. ระบบจัดการธุรกรรม (Transaction)
*   เพิ่ม (Add), แก้ไข (Edit), และลบ (Delete) รายการรายรับ-รายจ่ายได้
*   สามารถระบุหมวดหมู่ที่ต้องการ วันที่ และบันทึกโน็ตเพิ่มเติมได้ครับ
*   มีระบบกรองคำหยาบเบื้องต้นในช่องบันทึกโน็ตครับ

### 3. ระบบหมวดหมู่ (Category Management)
*   สร้างหมวดหมู่ส่วนตัวได้ เช่น ค่าอาหาร, ค่าเดินทาง, ค่าช้อปปิ้ง, ค่าน้ำมัน, และอื่นๆ อีกมากมายครับ
*   ช่วยให้สรุปยอดแยกตามหมวดหมู่ได้ชัดเจนครับ

### 4. ระบบสรุปข้อมูล (Summary)
*   ดูยอดรวมรายรับ รายจ่าย และยอดคงเหลือปัจจุบันทั้งหมดทุกบัญชีของผู้ใช้งาน
*   ดูสรุปตามช่วงเวลาต่างๆ ได้โดยการ filter ตามช่วงเวลาที่ผู้ใช้งานต้องการครับ

### 5. ระบบนำเข้าและส่งออกข้อมูล (Import & Export)
*   **Export:** สามารถดึงข้อมูลออกมาเป็นไฟล์ Excel, csv, JSON ได้ครับ
*   **Import:** สามารถอัปโหลดไฟล์เงินเข้า-ออกได้สะดวกโดยการอัพโหลดไฟล์ csv, Excel ได้ครับ (มีการตรวจรายการซ้ำก่อนบันทึกครับ)

## เทคโนโลยีที่ใช้
ในโปรเจกต์นี้ผมพยายามใช้เครื่องมือที่ผมเคยใช้และบางอย่างผมก้ไม่เคยใช้ครับ อาจจะมีถูๆ ไถๆ ไปบ้างครับ🥲
*   **Backend** Node.js + Express.js
*   **Database** MongoDB (ใช้สำหรับการเก็บข้อมูลแบบยืดหยุ่น)
*   **Authentication** การใช้ Token/Session สำหรับตรวจสอบสิทธิ์
*   **File Handling** Multer (สำหรับรับไฟล์อัปโหลด) และ XLSX (สำหรับสร้าง/อ่านไฟล์ Excel)

## วิธีรันโปรเจค
*   1. หลังจาก git Clone โปรเจ็คลงโฟล์เดอร์เรียบร้อยแล้ว
*   2. สามารถเข้า terminal แล้วรัน npm run dev / npm start ได้เลยครับ
